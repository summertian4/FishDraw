# 简介
iOS-FishDraw是一个简单的画图app。

动图展示:

![FishDraw效果演示](http://7xnrog.com1.z0.glb.clouddn.com/github_iOS-FishDraw-show.gif)

# 功能
1. 绘制椭圆、矩形、圆形等基本图形
2. 选择即将绘制图形的颜色
3. 缩放图形
4. 移动图形

# 结构
